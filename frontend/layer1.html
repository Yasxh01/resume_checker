How is this score calculated?

  • Smart Preprocessing: Automatically strips HTML tags and generic job boilerplate (e.g., "equal opportunity employer") to focus on core requirements.
  • Document Chunking: Long resumes are split into 400-word overlapping blocks to prevent context loss during neural embedding processing.
  • Hybrid Fallback: Combines dense vector similarity with TF-IDF bigram matching (e.g., catching specific phrases like "machine learning").
