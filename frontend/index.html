Explainable AI Pipeline

Five Independent AI Layers
One Transparent Hiring Decision

Every layer contributes measurable evidence instead of black-box scoring, allowing recruiters to understand why every candidate received their final ranking.

Layer 1: Semantic

Hybrid semantic matching using BAAI/bge-small-en-v1.5 embeddings, TF-IDF, chunked resume processing and intelligent caching.

Layer 2: Taxonomy

Alias-aware skill normalization with fuzzy matching and hierarchical taxonomy mapping.

Layer 3: Experience

Automatic employment timeline extraction, seniority inference and evidence-based experience validation.

Layer 4: Projects

Live GitHub repository analysis with language detection, topics, stars, forks and semantic fallback.

Layer 5: GitHub

Behavioral intelligence from contribution patterns, coding consistency and repository quality.

AI Interview Guide

Gemini-powered interview questions and hiring recommendations with rule-based fallback.
