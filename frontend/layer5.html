How is this calculated?

  • Contribution Patterns (40%): Fetches live PushEvents from the last 90 days to measure coding consistency (unique active days), overall commit volume, and commit discipline (average commits per push).
  • Code Quality Validation (30%): Aggregates Stars, Forks, documentation rates (README presence), and recent activity metrics across all repositories.
  • Repository & Follower Baseline (30%): Evaluates overall community reach and output using public repository counts and follower networks.
  • Developer Persona: Analyzes commit timestamps to determine if a candidate operates primarily as a "9-to-5 Corporate ☕" or a "Midnight Hacker 🦉".
  • Safe Fallback: Gracefully defaults to a neutral 50.0 score if no GitHub username is provided, ensuring candidates aren't artificially penalized for lacking open-source history.
